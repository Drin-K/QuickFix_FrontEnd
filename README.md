# QuickFix

Frontend setup for the QuickFix marketplace using React + TypeScript + Vite.

## Local setup

Requirements:

- Node.js 20+ (see `.nvmrc`)

Install + run:

```bash
npm install
npm run dev
```

## Project structure

```text
src/
  components/
  pages/
  layouts/
  contexts/
  hooks/
  utils/
```

## Scripts

```bash
npm install
npm run dev
npm run build
npm run typecheck
npm run lint
npm run format
```
